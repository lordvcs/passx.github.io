# PassX Website

This is the static landing page and privacy policy for the PassX mobile application.

## URL
The website is hosted at [https://passx.github.io](https://passx.github.io)

## Pages
- `index.html`: Landing page for PassX.
- `privacy.html`: Privacy Policy page required for Play Store and App Store submission.
- `style.css`: Common styles for the website.

## App Information
PassX is a smart sports and venue booking application that helps users find venues, book courts, and organize matches.
